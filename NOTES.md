# 一维中子扩散方程 — 项目讲解

> 从物理方程到 Python 代码的完整推导，覆盖单群和双群。

---

## 1. 物理背景：中子在堆芯里干什么？

```
        裂变
         ↓
  中子 → 撞击 U-235 → 释放 2-3 个新中子 + 能量
         ↓
  新中子 → 慢化（散射） → 热中子 → 再次裂变 → ...
```

核反应堆本质上是一个**中子增殖系统**。核心问题：

> **k_eff = 下一代中子数 / 当前代中子数**

- k > 1：中子越来越多，功率上升（超临界）
- k = 1：稳定运行（临界）
- k < 1：中子越来越少，停堆（次临界）

控制棒、硼酸都是用来调节 k 使之精确等于 1。

## 2. 中子扩散方程 — 从玻尔兹曼到 Fick 定律

中子输运的精确描述是玻尔兹曼输运方程，但直接解太难。做如下近似：

**Fick 定律**：中子流 J 正比于通量梯度
$$J = -D \nabla\phi$$

代入中子数守恒方程，得到**扩散方程**：

$$-D\nabla^2\phi(\mathbf{r}) + \Sigma_a\phi(\mathbf{r}) = \frac{1}{k}\nu\Sigma_f\phi(\mathbf{r})$$

| 项 | 物理含义 |
|---|---|
| $-D\nabla^2\phi$ | 泄漏项 — 中子从高浓度区域扩散出去 |
| $\Sigma_a\phi$ | 吸收项 — 中子被材料吸收 |
| $\frac{1}{k}\nu\Sigma_f\phi$ | 裂变源项 — 裂变产生的新中子 |

这本质上是一个**特征值问题**：求 k 使得方程有非零解。k 是特征值，φ 是特征向量（中子通量分布）。

---

## 3. 有限差分法 (FDM) — 把连续方程变成矩阵

### 3.1 一维离散化

把平板 [0, L] 切成 N 份，每份宽 h = L/N。网格点位于：
$$x_i = (i + 0.5)h,\quad i = 0, 1, \ldots, N-1$$

二阶导数用**中心差分**近似：
$$\frac{d^2\phi}{dx^2}\bigg|_{x_i} \approx \frac{\phi_{i-1} - 2\phi_i + \phi_{i+1}}{h^2}$$

### 3.2 组装成矩阵

代入扩散方程（单群）：
$$-\frac{D}{h^2}(\phi_{i-1} - 2\phi_i + \phi_{i+1}) + \Sigma_a\phi_i = \frac{1}{k}\nu\Sigma_f\phi_i$$

整理：
$$-\frac{D}{h^2}\phi_{i-1} + \left(\frac{2D}{h^2}+\Sigma_a\right)\phi_i -\frac{D}{h^2}\phi_{i+1} = \frac{1}{k}\nu\Sigma_f\phi_i$$

写成矩阵：**A φ = (1/k) F φ**

其中 A 是三对角矩阵：
```
       [2D/h²+Σ_a   -D/h²       0      ...]
A  =   [  -D/h²    2D/h²+Σ_a  -D/h²   ...]
       [    0        -D/h²    2D/h²+Σ_a ...]
       [   ...        ...       ...    ...]
```

边界条件 φ(0) = φ(L) = 0：直接成立，无需额外处理（因为网格点在半整数位置，边界点恰好位于 x=0 和 x=L）。

### 3.3 三对角矩阵的特征值 — 解析公式

N×N 矩阵 (2, -1, -1) 的特征值和特征向量**解析已知**：
$$\lambda_k = 4\sin^2\left(\frac{k\pi}{2(N+1)}\right),\quad k=1,\ldots,N$$
$$\mathbf{v}^{(k)}_j = \sin\left(\frac{k j \pi}{N+1}\right),\quad j=1,\ldots,N$$

加上 D/h² 和 Σ_a：
$$\alpha_k = \frac{4D}{h^2}\sin^2\left(\frac{k\pi}{2(N+1)}\right) + \Sigma_a$$

最小的 α_k（k=1）对应最平坦的正弦波 → **基模**（fundamental mode），是反应堆的稳态通量形状。

这个解析公式就是我们 debug 的核心工具——用来验证代码矩阵是否正确构造。

## 4. 幂迭代 (Power Iteration) — 不直接求特征值

直接对 N×N 矩阵求特征值（`np.linalg.eigvals`）在小规模可行，但实际问题 N 可达 10⁵~10⁷，必须用迭代。

### 4.1 算法

```
初始化 φ = [1, 1, ..., 1]

每步迭代：
  1. s = F @ φ              ← 当前裂变源
  2. φ_new = solve(A, s)    ← 解线性方程组，得新通量
  3. k = sum(F @ φ_new) / sum(s)  ← 新/旧 裂变中子数之比
  4. φ = φ_new / max(φ_new) ← 归一化（防溢出）
```

步骤 2 的核心是 `np.linalg.solve(A, s)`，它求解线性方程组 A·x = s。由于 A 是三对角的，实际工程中会用 Thomas 算法（O(N) 而非 O(N³)）。

### 4.2 为什么 k 的公式必须是裂变中子比？

**物理定义**：k = 下一代裂变产生的中子数 / 当前代裂变产生的中子数

即 k = Σ(νΣ_f · φ_new) / Σ(νΣ_f · φ_old) = Σ(F·φ_new) / Σ(F·φ_old)

**错误公式**（原代码）：`k_new = k_old * sum(φ_new) / sum(φ)`

为什么错？收敛时 φ_new = k_true · φ，代入：
```
k_new = k_old * sum(k_true · φ) / sum(φ) = k_old · k_true
```

要求 k_new = k_old，则 k_true = 1。但实际 k_true ≠ 1，所以永远不收敛——k 会指数衰减到 0。

**正确公式**：收敛时：
```
k_new = sum(F · k_true · φ) / sum(F · φ) = k_true ✓
```

## 5. 双群扩散 — 区分快中子和热中子

### 5.1 为什么需要多群？

单群把所有中子当成同一能量，但实际物理中：

```
快中子 (0.1 MeV)  →  散射慢化  →  热中子 (0.025 eV)  →  裂变
    ↑                                                ↑
  少量裂变                                     主要裂变来源
```

群 1（快群）：高能中子，泄漏多，主要经历散射慢化
群 2（热群）：低能中子，泄漏少，是裂变的主力

### 5.2 耦合方程

**群 1（快群）：**
$$-D_1\nabla^2\phi_1 + \underbrace{(\Sigma_{a1} + \Sigma_{s,1\to2})}_{\Sigma_{r1}}\phi_1 = \frac{1}{k}[\nu\Sigma_{f1}\phi_1 + \nu\Sigma_{f2}\phi_2]$$

- Σ_r1 = 移出截面 = 吸收 + 散射到群 2
- 裂变源来自两个群（快中子也会引起裂变，但贡献小）

**群 2（热群）：**
$$-D_2\nabla^2\phi_2 + \Sigma_{a2}\phi_2 = \Sigma_{s,1\to2}\phi_1$$

- 右端是散射源——"快群喂给热群的中子"
- 热群不直接裂变产生中子（在矩阵中 F₂₁ = F₂₂ = 0）

### 5.3 分块矩阵结构

把 [φ₁, φ₂] 拼成 2N 维向量：

```
┌──────────┬──────────┐ ┌──┐       ┌──────────┬──────────┐ ┌──┐
│  A₁₁     │    0     │ │φ₁│       │  F₁₁     │  F₁₂     │ │φ₁│
│          │          │ │  │ = 1/k │          │          │ │  │
│  A₂₁     │  A₂₂     │ │φ₂│       │    0     │    0     │ │φ₂│
└──────────┴──────────┘ └──┘       └──────────┴──────────┘ └──┘

A₁₁ = coeff1·L + Σ_r1·I      (快群：泄漏 + 移出)
A₂₁ = -Σ_{s,1→2}·I           (耦合！群1→群2 散射)
A₂₂ = coeff2·L + Σ_a2·I      (热群：泄漏 + 吸收)

F₁₁ = νΣ_f1·I,  F₁₂ = νΣ_f2·I  (两个群都贡献裂变)
```

注意：A₂₁ 带负号是因为把散射源移到了左边：
$$-D_2\nabla^2\phi_2 + \Sigma_{a2}\phi_2 - \Sigma_{s,1\to2}\phi_1 = 0$$

### 5.4 截面数据的物理含义

| 参数 | 群 1 (快) | 群 2 (热) | 解释 |
|------|----------|----------|------|
| D | 1.2 cm | 0.4 cm | 快中子扩散更快 |
| νΣ_f | 0.003 | 0.105 | 热中子裂变概率远大于快中子 |
| Σ_a | 0.008 | 0.08 | 热中子更容易被吸收 |
| Σ_{s,1→2} | 0.020 | — | 慢化截面 |
